# StockScope - 股票數據視覺化儀表板

一個專業的股票數據視覺化儀表板，具備公允價值量規和財務趨勢圖表。

## 主要功能
- **股票視覺化**：收入、淨利和利潤率的互動式圖表。
- **公允價值量規**：即時分析股票價值（強力買入、合理買入、持有、估值過高）。
- **效率指標**：ROE、FCF 收益率、流動比率等。
- **數據導入**：可輕鬆通過 JSON 導入股票數據。
- **比較工具**：比較不同股票的本益比 (P/E)、利潤率和增長率。

## 技術棧
- React 19
- Tailwind CSS
- Recharts
- Lucide React
- Motion (Framer Motion)

## 如何在本地運行
1. **下載程式碼**：您可以點擊 AI Studio 介面右上角的 **Settings** (齒輪圖示) -> **Export to ZIP** 來下載整個專案。
2. **安裝依賴**：
   ```bash
   npm install
   ```
3. **啟動開發伺服器**：
   ```bash
   npm run dev
   ```
4. **訪問應用**：在瀏覽器中打開 `http://localhost:3000`。
